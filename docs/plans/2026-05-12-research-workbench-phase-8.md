# Phase 8：前端投研工作台实施记录

## 目标

把前面阶段已完成的数据、信号、报告、回测、审查、调优能力接入 React 前端，形成可日常使用的 A/H 股个人投研工作台。

## 已实现范围

- 新增工作台导航，保留旧单股分析入口。
- 新增自选股页面，支持批量添加、市场识别、行业、关注逻辑和移除。
- 新增今日信号页面，按重点观察、新增观察、等待确认、风险升高、信号失效分组。
- 新增每日复盘页面，支持 Markdown 表格渲染和下载。
- 新增事件回测页面，展示统计表和失败样本。
- 新增信号审查页面，展示 action、置信度、理由、风险旗标和缺失数据。
- 新增策略调优页面，展示诊断报告和候选 YAML。
- 新增数据健康页面，展示质量检查记录。
- 新增 `/api/research/*` 路由，覆盖自选股、数据质量和策略调优诊断。

## 验收口径

- 不接真实交易接口。
- 前端新增页面不输出禁止文案。
- 所有新增 API 测试不依赖外部网络。
- 生产构建和后端测试必须通过。
